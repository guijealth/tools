| Command            | Keybindings         |
|--------------------|---------------------|
| wrap around parens | `ALT + (` `ALT + )` |
| wrap around square | `ALT + [` `ALT + ]` |
| wrap around curly  | `ALT + {` `ALT + }` |
| wrap around quote  | `ALT + "`           |
| next or up sexp    | `L`                 |
| prev or up sexp    | `H`                 |
| slurp forward      | `>`                 |
| barf forward       | `<`                 |
| slurp backward     | `ALT + <`           |
| barf backward      | `ALT + >`           |
| kill sexp forward  | `ALT + d`           |
